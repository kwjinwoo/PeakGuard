# Roadmap

The roadmap records direction without depending on an external issue tracker.

- [Now](now.md): work that protects correctness or clarifies the current system.
- [Next](next.md): likely improvements after the current baseline is stable.
- [Later](later.md): useful ideas without near-term commitment.

Items describe outcomes and rationale, not detailed implementation plans. Move an item between pages when its priority changes. Accepted constraints belong in an ADR; exploratory designs belong in proposals.
