# Later

## Validate configuration-to-document consistency

Consider a lightweight check that detects when documented assets or configuration examples drift from `portfolio.yaml`.

## Improve report extensibility

If additional signals are adopted, consider a typed signal representation so adding a metric does not require parallel boolean fields throughout orchestration and formatting. Avoid this abstraction until another implemented signal justifies it.

## Add Wiki integrity checks

Consider local link validation and optional front-matter validation once document volume makes manual review unreliable.
