# Architecture Decision Records

Architecture Decision Records (ADRs) preserve the rationale for choices that constrain future PeakGuard changes.

## Index

- [ADR-0001: Persist rolling history as CSV in a GitHub Gist](0001-csv-gist-persistence.md)

## Template

Copy the [ADR template](../meta/templates/adr.md) and assign the next zero-padded number.

An ADR describes why a decision exists; current operational behavior belongs in the topical wiki pages. When superseding an ADR, retain the old record and link both decisions.
